Creating multi-stage blends

Xara LX lets you create blends using as many objects as you like.

 To create a multi-stage blend

1. Choose the Blend Tool.

2. Create a blend from the first object to the second (see Blending objects).

3. Drag from the blend to the third object.

This creates a multi-stage blend between the three objects.

Tips

 The multi-stage blend always runs from the second object to the third. (The first object is where you started dragging from.) If the multi-stage blend is not correct, remove the blends and retry. See Removing a blend.

 You are not limited to three objects in a multi-stage blend. You can continue to blend to further objects.

 You can blend two blends together. This is called a two-way blend. See Creating two-way blends for more details.
