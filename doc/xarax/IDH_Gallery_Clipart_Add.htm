Clipart Gallery - Disk Clipart Button

 

The Disk Clipart button in the Clipart Gallery opens a dialog box which you can use to choose a folder to add to the gallery or update an existing folder. This would usually be used to open the clipart folder on the Xara LX CD, though any folder containing clipart can be added to the gallery. See Adding clipart to the clipart gallery for more details.
