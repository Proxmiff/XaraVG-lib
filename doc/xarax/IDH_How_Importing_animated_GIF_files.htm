Importing animated GIF files

If you import or open an animated GIF file, each frame is imported as a bitmap with all the bitmaps placed in separate frames. Each frame is listed in the Frame Gallery with the last frame at the top.

Tip

 You can easily alter an animated GIF by loading it into Xara LX, altering one or more frames and reexporting the animated GIF. For more details, see Creating animated GIF files.
