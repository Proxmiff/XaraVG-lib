Editing text

There are two editing modes used with the Text Tool. You can either change the text using the text cursor or by changing a selected region of text

 To edit text with the text cursor

1. Choose the Text Tool.

2. Click in the text object to insert the text cursor.

3. Edit the text as required (see List of Text Tool Key Shortcuts).

 To edit text selections

When a region of text has been selected (see Selecting text) you can:

 Apply attributes to the selected region - see Applying attributes to text

 Replace the selected region by typing new text.

 To scale and stretch text

You can use either the Text Tool (see Changing the size of text or Changing the aspect ratio of text) or the Selector Tool (see Scaling objects or Stretching objects).

For information on changing the width of a column text object, see Resizing column text.

Tip

 Whenever the text cursor is moved it takes on the attributes of the character to its left.
