Bitmap Gallery - Insert Button

The Insert button in the Bitmap Gallery inserts the currently selected bitmap into the selected document. Any bitmap in the gallery can be inserted into the selected document, even from other sections of the gallery. This means this button can be used to copy bitmaps between documents.

You can also drag-and-drop the bitmap onto a blank part of the page to insert it or open the pop-up menu over the bitmap and choose Insert. Note that if you drop the bitmap onto an object it will be applied as a bitmap fill - drop it on a blank area of the page to insert it.
