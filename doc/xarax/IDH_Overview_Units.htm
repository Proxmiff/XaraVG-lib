Units Overview

Xara LX has a high level of support for units of measurement. Units are used widely throughout the program and you can define additional custom units as required (see Creating a custom unit). Whenever a field is visible showing a measurement in units, you can type in a value using any unit. For example, you can type "1pt" or "2cm" or even "10mi" into the Font Size field on the Text Tool Infobar.

The default Xara LX units are:

pix  Pixels (1/96 inch)

mm  Millimeters

cm  Centimeters

m  Metres

in  Inches

ft  Feet

yd  Yards

pt  Points (1,000 mp or 1/72 inch)

pi  Picas (12,000 mp 1/6 inch)

mp  Millipoints (1/72,000 inch)

mi  Miles

km  Kilometers

A millipoint is 1/72000 in. and is used for all internal measurements.

Tip

 You can use a scale factor with your document. For example you could draw a map at a scale of 25,000 to 1 with the units appearing on the Status Line and Infobar in Kilometers. See Setting a scale factor.
