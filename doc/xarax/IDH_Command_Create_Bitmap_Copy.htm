Create Bitmap Copy (Arrange Menu) CTRL+Shift+C

Create Bitmap Copy creates a bitmap version of the selected objects. When you select it, it opens the Create Bitmap Copy dialog box where you can set the options for the bitmap and even preview it before you insert it into the document. Click the Create button to create the bitmap. By using the Transparent Background option you can create non-rectangular bitmaps. For more information on creating bitmaps from objects, see Creating bitmaps from objects.
