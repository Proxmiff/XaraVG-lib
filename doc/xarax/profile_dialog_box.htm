Profile dialog box

This is used by several tools to control attribute or color transitions.

You can either:

 select a predefined profile from the drop-down list - the top option (the straight line) produces a linear transition

 or create a custom transition by dragging the sliders (or type in values). The display and the strip shows the resultant profile. For example for a color transition, the display shows how the colors change with the selected slider settings.

Move the top slider towards:

 -1 for more change at the end of the transition than at the start.

 +1 for more change at the start.

Move the bottom slider towards:

 -1 for more change in the center than at the ends.

 +1 for more change at the ends.
