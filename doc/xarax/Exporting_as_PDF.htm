Exporting as PDF

PDF is an ideal format to exchange and distribute files, in particular for downloading information on websites and providing your work to commercial printers.

The PDF exporter has also been designed to allow the PDF files to be opened by and edited in Adobe Illustrator, so providing Illustrator compatibility. Vector artwork, including graduated color and transparency, can be imported cleanly into Illustrator version 9 or later.

To export as PDF, select File then Export. Under 'save as type' select the PDF option.
