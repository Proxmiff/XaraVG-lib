Ensuring the current layer is always editable

When you select a layer in the Layer Gallery as the current layer (the one objects will be created in) you may find the current layer has been set so you cannot edit it. You can change this so the current layer is always editable and visible.

 To ensure the current layer is always editable

1. From the Utilities Menu, choose Options (see Options).

2. Click the General tab.

3. Set the Current Layer Always Visible and Editable option.
