Bitmap Effects - Resize

To change the size of an image in Xara LX,  it is easiest to use the Selector Tool (see Scaling Objects) and dragging the corners of the bitmap to change it to the required dimensions.

Iin some cases you may need to alter the actual bitmap. To create a bitmap of a different size or to change the resolution can you can create a bitmap copy by selecting the bitmap and Ctrl-Shift C or selecting Create Bitmap Copy from the Arrange menu. You can then select the required size.

Tips

 To keep the bitmap the same shape, select the Lock Aspect Ratio option.

 Creating bitmap copies will increase the file size because new bitmaps are not stored using JPEG compression.
