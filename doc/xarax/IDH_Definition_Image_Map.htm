Image maps are used in HTML pages to create several clickable areas over images. Clicking on one of these areas takes you to another location on the Internet. For example, if you had a picture of the Earth, you could use an image map with a clickable area over each different continent. Clicking on the clickable area for a continent could take you to a page of information about that continent. The clickable areas can be rectangles, circles or multi sided irregular polygons. Xara LX supports client-side image maps (it doesn't support server-side image maps). For more details on image maps, see the Image Maps Overview.
