Button & NavBar Tool - Edit States

This lets you change single-state buttons into multistate rollover buttons. The existing button design becomes the MouseOff state button. Clicking the button opens the Edit States dialog box, which lets you create and delete MouseOver, MouseDown and Selected button states. (Rollovers create more complicated HTML code on the web page. If you are an Internet novice, we suggest using only simple bars.)
