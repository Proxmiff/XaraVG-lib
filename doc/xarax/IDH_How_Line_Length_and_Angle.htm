Setting the length and angle of a straight line segment

When you have drawn a line or shape that includes straight segments, you can set the angles of these segments exactly.

 To set the length and angle of a segment

1. Select the line or shape (see Selecting objects).

2. Choose the Shape Editor Tool.

3. Select the point handle on one end of the segment.

4. Enter the angle and length into the fields on the Shape Editor Tool Infobar.

5. Press RETURN.
