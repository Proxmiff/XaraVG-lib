Changing the page size

You can produce drawings on virtually any size page.

 To change the page size

1. From the File menu, choose Page Options (see Page Options).

3. From the Paper Size drop-down list, select the new page size. For example A4.

Tips

 For details of changing the paper size for all new documents, see Changing the default paper size.

 If you want to use a paper size not listed, select the Custom option from the drop-down list and enter the width and height
