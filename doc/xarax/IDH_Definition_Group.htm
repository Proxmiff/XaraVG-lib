Group objects are compound objects which contain other objects (even groups). The group object is a single object and can be treated like any other, for example applying attributes to it, rotating it, etc. Note that if you want to change one object in the group you do not have to ungroup it, you can use select inside to select it from within the group. For more details, see Grouping objects.
