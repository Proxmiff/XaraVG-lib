Adding shapes

You add objects together to combine then into a single shape. The new shape covers the same area as all the objects you added.

 To add objects together

1. Select the objects (see Selecting objects).

2. From the Combine Shapes submenu on the Arrange Menu, choose Add Shapes (or press CTRL+1).

Tips

 The new shape uses the colors, line width and other attributes of the top shape. You can change the stacking order of objects. For details see Moving objects backwards and forward.

 You can use all object types including bitmaps.
