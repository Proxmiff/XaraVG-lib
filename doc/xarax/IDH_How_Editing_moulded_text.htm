Changing molded text

Text which has been molded cannot be edited directly in Xara LX, but described below is a simple process which allows you to easily remove the mold, edit the text and reapply the mold.

 To edit molded text

1. Select the molded text object (see Selecting objects).

2. Choose the Mold Tool.

3. Click the Copy Mold button on the Mold Tool Infobar.

4. Click the Remove Mold button on the Infobar to remove the mold from the text.

5. Edit the text object as normal (see Editing text.)

6. Choose the Mold Tool again.

7. Click either Paste Envelope or Paste Perspective button on the Infobar.

Tips

 Don't use Cut or Copy while you're editing the text as this overwrites the mold on the clipboard.

 You could make it easy to change the colors in the text by using named colors. You can then change the colors without having to remove the mold.
