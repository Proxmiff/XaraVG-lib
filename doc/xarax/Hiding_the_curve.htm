Hiding the line in blend on a curve

When fitting a blend to a curve it is useful to have the line or shape visible so you can edit the shape of the curve if required. When you are happy with the results you can make the line invisible.

 To hide the line in blend on a curve

1. Select the blend on the curve (see Selecting objects).

2. Choose the Shape Editor Tool. This selects just the line or shape.

3. Shift-click (or right-click) on the No color button on the Color Line.
