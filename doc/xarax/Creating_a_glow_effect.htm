Creating a glow effect

Creating a glow or halo behind an object can create dramatic graphics.

 To create a glow effect

1. Select the object (see Selecting objects). (See Note.)

2. Choose the Shadow Tool.

3. Select the Glow button on the Shadow Tool Infobar.

You can alter the shadow to suit your needs - see Modifying shadows.

Note:

 You can also use the Shadow Tool to select objects - more details.
