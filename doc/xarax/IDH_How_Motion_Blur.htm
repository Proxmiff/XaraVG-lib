Drawing motion blur

Drawing motion blur with Xara LX is easy!

 To draw motion blur

1. Draw the object to be blurred (see Drawing shapes).

2. Clone the object (see Copying objects).

3. Apply a flat transparency of (for example) 85% to the clone (see Applying flat transparency).

4. Clone the transparent object.

5. Move the new object in the blur direction (see Moving objects).

6. Alter the transparency of the second object to be almost 100%.

7. Blend between the two transparent objects (see Blending objects).

8. Increase the number of blend steps as required (see Changing the number of steps in a blend).
