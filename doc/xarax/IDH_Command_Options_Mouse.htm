Mouse Options

To open the Mouse Options, choose Options from the Utilities menu and click on the Mouse tab.

Left and Right Mouse Button Function

The option controls what the left mouse button does. For more information on configuring the mouse, see Configuring the mouse buttons.

Magnetic Snap Radii

These two fields set the magnetic snap distances for grid, guide and object snapping. For more information, see Changing the magnetic snap distances.
