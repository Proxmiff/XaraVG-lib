Modifying shadows

The Shadow Tool creates soft shadows, which fade out to totally transparent at the edges. This looks more realistic than a solid shadow.

What do you want to do?

 Blur the edges of the shadow (change the size)

 Change the size or direction of a shadow

 Move a Wall shadow

 Change the transparency of the center of the shadow

 Change the shadow profile

 Change the color or fill effect of the shadow

 Create a glow effect

 Convert to editable shapes (for advanced editing)

Note:

 Shadows created with the Shadow Tool use bitmap transparency to create the shadow effect. This means you cannot use the Transparency Tool to apply a transparency type to a shadow.
