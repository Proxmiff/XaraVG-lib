Color Editor - Component Values

These fields define the values for the components of the color. The components shown depend on the color model in use. Note that when linked colors, tints and shades are being edited, some of these fields are dimmed and cannot be altered directly.

The values can be edited in one of two units, percentages (0% to 100%) or values (0 to 255). Which is used is set by the Color Units setting in the Units Options. If you need to enter a value in the other format, you can force the units by typing (for example) "25%" (for a percentage) or "25d" (for 0 to 255 value). There are two exceptions: (a) Hue in the HSV color model is an angle around the color circle and (b) the RGB Web color model defines colors as six hex digits (0-9, a-f).
