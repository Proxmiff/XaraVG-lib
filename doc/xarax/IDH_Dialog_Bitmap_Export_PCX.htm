PCX Export Dialog Box

The PCX export bitmap dialog box is opened by selecting PCX as the exported file format from the Export dialog box. It allows the setting of the size/resolution of the exported bitmap, the area to be exported and the number of colors in the bitmap. For details of how to create a PCX file, see Creating PCX files.
