Warping the edges of polygons

You can curve the sides of polygons created using the QuickShape Tool.

 To warp the edges of a polygon

1. Select the polygon (see Selecting objects).

2. Choose the QuickShape Tool

3. Move the mouse pointer over a side of the QuickShape. (The pointer changes to an arrow.)

4. Drag to reform the sides of the polygon.

Tips

 The sides of a QuickShape can be made straight again using the Restore Edges Button on the QuickShape Tool Infobar.

 If you hold down CTRL as you drag the edges of a starred polygon, both sides of each star point will move in parallel.

 If you hold down CTRL+Shift as you drag the edges of a starred polygon, both sides of each star point will move symmetrically.
