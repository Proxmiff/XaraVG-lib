Name Gallery - Pop-up Menu

Right-clicking on the Name Gallery opens a pop-up menu which contains the following options:

Find   Lets you find items in the Gallery. For more details, see Searching a gallery.

Apply  Identical to the Apply Button

Remove  Identical to the Remove Button

Redefine  Identical to the Redefine Button

Select  Identical to the Select Button

Intersect  Identical to the Intersect Button

New  Identical to the New Button

Rename  Identical to the Rename Button

Delete  Identical to the Delete Button

Export  Identical to the Export Button

(Un)Fold Section Folds up or unfolds the current section

Previous Section Moves to the start of the previous section

Next Section Moves to the start of the next section
