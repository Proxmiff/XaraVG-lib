Paste Special Dialog Box

If when you Paste with data on the clipboard from another application and there is more than one format that the data could be transferred as, the Paste Special dialog box will appear giving you a chance to choose the specific format that you require Xara LX to import. The highlighted format (always the top item in the list) is the one which is considered the "best" format to paste.

Choose the format which you prefer to paste, and then click the Paste button. Alternatively, you may click Cancel to cancel the paste operation entirely.

Note

 Once you have pasted something into a document, the pasted data is remembered in order to make subsequent pastes faster (as importing some clipboard data formats can take some time). This means the Paste Special dialog box will not appear again until you copy new data to the clipboard from another application.
