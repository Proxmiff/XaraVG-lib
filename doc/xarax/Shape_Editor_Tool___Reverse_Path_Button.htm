Shape Editor Tool - Reverse Path Button

This button on the Shape Editor Tool Infobar reverses the direction of the selected line. (So the start of the line becomes the end and vice versa.) Note that the shape of the line does not change, just its direction. Reverse Path has two uses:

 When you have an arrowhead applied, you can quickly move it to the other end of the line.

 To reverse the direction of a brush stroke; some brush strokes are 'directional' and this lets you reverse that direction.
