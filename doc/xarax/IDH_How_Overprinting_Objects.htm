Overprinting objects

Overprint can be used in multicolor work that will be litho-printed. See the Overprinting Overview for details of when overprinting is useful.

 To apply overprint to an object

This is the most useful form of overprint.

1. Select the object (see Selecting objects).

2. On the Utilities menu open the Imagesetting submenu then choose either

 Overprint Fill

 Overprint Line

Overprinting the fill is most commonly used to overprint dark colored text. Overprinting the line overcomes possible mis-registration problems. See the Overprinting overview for more information on mis-registration. To cancel object overprint, click Overprint Fill or Overprint Line to unselect.

Tips

 Do not apply overprint to large areas as this can flood the printing press with too much ink.

 Black ink is relatively opaque and overprints well. Other ink colors can show a color shift when overprinted.
