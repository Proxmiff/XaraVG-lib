Open (File Menu) CTRL+O

Opens an existing document or file in a supported format in a new window. Xara LX can open a wide variety of file formats, for a full list, see the List of Supported File Formats. For information on how to open files, see Opening documents from disk.

If you click on a PNG, BMP, GIF, JPEG, XAR or WEB file, you see a preview of it in the dialog box. (WEB files: you see a preview only if the file contains a preview - this is a Save option).
