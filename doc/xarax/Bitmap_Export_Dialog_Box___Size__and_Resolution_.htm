Bitmap Export Dialog Box - Size (and Resolution)

This section is not present if you are exporting a bitmap from the Bitmap Gallery. If you are exporting an existing bitmap, you cannot resize it.

Bitmap Size and Resolution

The size of the exported bitmap can be set in two ways:

Resolution Type the resolution into the dpi field. The Width and Height are calculated from this to ensure the aspect ratio of the area being exported is fixed.

Size  Type the required width or height into one of the fields. Note that the aspect ratio of the drawing is fixed so only one dimension can be set - the other is calculated automatically.

Tip

 Choosing a resolution of 96dpi ensures the bitmap is the same size as the objects on screen (at 100% magnification).
