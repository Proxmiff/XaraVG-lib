Removing a blend

If a blend does not give the desired effect, you can easily remove it.

 To remove a blend

1. Select the blend object (see Selecting objects).

2. Choose the Blend Tool.

3. Click on the Remove button on the Blend Tool Infobar.

Tip

 You can edit either of the end objects without having to remove the blend. See Select inside for details of how to select an end object in a blend.
