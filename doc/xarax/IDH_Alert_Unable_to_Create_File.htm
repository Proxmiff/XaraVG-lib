Message: Cannot create file

Cause

The program tried to create a file but did not succeed. This may be because:

 The disk is full.

 The drive is read-only (For example, a CD-ROM drive).

 A file of that name already exists and is read-only.

 You do not have write permission for the folder.

What Can I Do?

 Check that the disk is not full - if it is, then choose another disk or delete some unwanted files from it.

 Choose another name for the file, or another disk or folder.

 Ask your system administrator to change your access permissions.
