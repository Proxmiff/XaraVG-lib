Selecting how colors appear in the Color Editor

The color editor usually shows the HSV color editor when editing local colours, irrespective of whether they are RGB or CMYK colors

 

This is because the HSV color editor is more intuitive to use.

 

You can alter this behaviour so that it shows the CMYK or RGB modes when editing CMYK or RGB colors. Select the menu Utilities -> Options. In the View tab select

the 'Edit local colors in -> Automatic colour model'. Now when you select an object, if its color is CMYK then the color editor will show a CMYK controls.
