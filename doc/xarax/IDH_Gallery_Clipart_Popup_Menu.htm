Clipart Gallery - Pop-up Menu

Right-clicking on the Clipart Gallery opens a pop-up menu which contains the following options:

Open   Identical to the Open button.

Import  Identical to the Import button.

Remove Section Identical to the Remove button.

(Un)Fold Section Folds up or unfolds the current section.

Previous Section Moves to the start of the previous section.

Next Section  Moves to the start of the next section.

Note

Right-clicking in the gallery selects the picture under the mouse pointer, so using the pop-up menu is a quick way to select a picture and operate on it in one go.
