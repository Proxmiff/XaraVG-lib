Changing curved corners on a rectangle

Rectangles created using the QuickShape Tool can have either sharp or rounded corners.

 To change the corner curvature of a rectangle

1. Select the rectangle (see Selecting objects).

2. Choose the QuickShape Tool or the Rectangle Tool.

3. Drag one of the curvature handles. The QuickShape corners become more (or less) curved as you drag the handle.
