Options (Utilities Menu)

CTRL+Shift+O

Options on the Utilities menu brings up the Options dialog box. There are two types of option you can set:

 Program options

 Options for the current document

When options apply only to the current document, the section of the Options dialog box will show the document name (for example 'Grid options for map.xar'). The program options are saved when you finish using Xara LX whereas the document options are saved with the document.

If you want to set the default options, see Changing the template document.

The options are divided into the following sections:

 General Options

 Grid and Ruler Options

 Internet Options

 Mouse Options

 Page Options

 Effects & Plugins Options

 Scaling Options

 Tune-ups Options

 Units Options

 View Options
