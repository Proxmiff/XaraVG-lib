Applying attributes to individual text characters

You can easily apply attributes to a single character inside a text object.

 To apply attributes to a single character in a text object

1. Select the text character (see Selecting text).

2. Apply the attributes. For example, choose the Text Tool and change the font size or click on a color on the Color Line.

Tip

 You can also apply attributes to a single character by holding CTRL and using drag-and-drop. For example, dragging a fill from the Fill Gallery and dropping it on a character whilst holding down CTRL will apply a bitmap fill to one character rather than the whole text object. See apply inside for more details on applying attributes inside objects.
