Moving between documents

You can have several Xara LX documents loaded in memory at the same time. You can then view any of them in the window and swap between them.

 To move between documents

At the bottom of the Window menu is a list of currently loaded documents. From this list select the name of document you want to view.

Tip

 You can also cycle through the currently loaded documents by pressing CTRL+Tab.
