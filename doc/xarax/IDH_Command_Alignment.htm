Alignment (Arrange Menu)

CTRL+Shift+L

You can use the Alignment dialog box to align and distribute the selected objects The two drop-down lists allow you to set how the objects are moved horizontally and vertically. The diagram at the top shows how the objects will move.

The options in the 'Within' section allow you to set whether the objects move relative to the boundary of the selected objects (the bounding box), the page or the spread (a pair of pages). For example, if you select "Align Left" and have the Within Page(s) option set, the objects move so their left edges all rest on the left hand side of the page. For more details, see Aligning objects.

You can quickly align the selected objects to an edge or corner by clicking in the upper part of the dialog box.

Horizontal Positions

No Change leaves the objects in the same horizontal position.

Align Left moves the objects so their left hand edges are aligned vertically.

Align Center moves the objects so their centers are aligned vertically.

Align Right moves the objects so their right hand edges are aligned vertically.

Distribute Left Edges moves the objects so their left edges are evenly spaced horizontally.

Distribute Centers moves the objects so their centers are evenly spaced horizontally.

Distribute Right Edges moves the objects so their right hand edges are evenly spaced horizontally.

Distribute Equi-spaced moves the objects so the spacing between them is even horizontally.

Vertical Positions

No Change leaves the objects in the same vertical position.

Align Bottom moves the objects so their bottom edges are aligned horizontally.

Align Centrally moves the objects so their centers are aligned horizontally.

Align Top moves the objects so their top edges are aligned horizontally.

Distribute Bottom Edges moves the objects so their bottom edges are evenly spaced vertically.

Distribute Centers moves the objects so their centers are evenly spaced vertically.

Distribute Top Edges moves the objects so their top edges are evenly space vertically.

Distribute Equi-spaced moves the objects so the spacing between them is even vertically.
