Create a ClipView

ClipView lets you use an object as a "window" onto other objects. You see only those parts of other objects that are within the window. We call this creating a ClipView.

 To create a ClipView

1. The window object must be behind the other objects in the ClipView. If necessary move it backwards. In this example the blue ellipse is the window object:

 

2. Select all the objects you want in the ClipView - see Selecting objects.

3. Choose Apply ClipView on the Arrange menu. All the window object remains visible plus those parts of other objects that cover the window object.
