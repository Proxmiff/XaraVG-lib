Name Gallery - Export Button

Reexport the selected named objects. To select a named object for export click its name, not the circle to the left. (Clicking the circle selects the objects in the document, not the name in the gallery.) You can select multiple names by clicking the first name then CTRL-clicking the other names.

You can change the export options by selecting Exports from the drop down list - more information on the options.
