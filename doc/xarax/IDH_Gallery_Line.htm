Line Gallery Overview

F12

The Line Gallery gives full control over the appearance of lines and shape outlines. It allows you to alter dash pattern, width, joins and ends.

The line attributes that can be set using the gallery are:

  Line Width

 Join Type

 Line Cap

    Brush Strokes (arranged in two or more sections)

    Stroke shapes

    Dash Pattern

    Arrows

The Brush Strokes sections have a pop-up menu with options of Rename brush, Export shapes to clipboard and (New brush stroke section only) Delete. More details in Brush Strokes.
