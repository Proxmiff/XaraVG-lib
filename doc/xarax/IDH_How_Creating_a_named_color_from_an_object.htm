Creating named colors from objects

If you have an object in your document which is filled with a color, you can create a named color using that color.

 To create a named color from an object

1. Select the object filled with the required color (see Selecting objects).

2. From the Utilities menu, choose Color Editor.

3. In the Color Editor, click the New Named Color button to open the New Named Color dialog box.

4. Deselect Make color a shade of. (You do not want to create a shade color.)

5. Enter a name for the new color.

6. Click Create. The new named color appears on the Color Line and in the Color Gallery.

The color is also displayed in the Color Editor so you can edit it as required. See Changing named colors.

Tip

 If you want to change your new named color, just drag it onto the Color Editor.
