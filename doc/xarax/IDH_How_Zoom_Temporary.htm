Using the Zoom Tool temporarily

The Zoom Tool can be used temporarily - alleviating the need to manually choose the previous tool afterwards.

 To use the Zoom Tool temporarily

1. Press and hold down ALT+Z to swap to the Zoom Tool.

2. Use the Zoom Tool as required.

3. Release ALT+Z to return to the previous tool.

or

1. Press and release ALT+Z to swap to the Zoom Tool.

2. Use the Zoom Tool as required.

3. Press and release ALT+Z again to return to the previous tool.
