Pen Tool Shift+F5

The Pen Tool is an alternative way of creating lines and shapes. We recommend using the Shape Editor Tool as it lets you create and edit lines using the same tool. The Pen Tool operates in exactly the same way as equivalent tools in other programs such as CorelDRAW or Adobe Illustrator. If you are familiar with such programs, you may prefer to use this tool to draw lines and shapes. To find out more about lines and shapes, see the Lines and Shapes Overview.
