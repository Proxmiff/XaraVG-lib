Infobar Overview

The Infobar displays different information and controls dependent on the currently selected Tool. For example, when you choose the Text Tool the Infobar contains only text related controls and when you choose the Zoom Tool, the Infobar only shows zoom related controls. By doing this, you can easily select which buttons are available and only ever have one set visible on screen - no more screen clutter!

For more information on each of the tools' Infobars, see The Tools.
