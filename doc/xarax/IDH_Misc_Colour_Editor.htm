Color Editor (Utilities Menu) CTRL+E

The Color Editor is used to edit the color of objects (local colors) and to create and change named colors. It allows you to edit colors using a range of color models and even allows you to create colors which are linked to other colors (their 'parent' color) so that when the parent color changes, all connected colors will also change - see Color Editor - Advanced. For information about colors in Xara LX, see the Colors Overview.

You can open the Color Editor by:

 Selecting Color Editor from the Utilities menu or the selection pop-up menu.

 Clicking the Edit Color button on the Color Line to edit the current fill color (Shift-click or Right-Click for line color)..

 Selecting the Edit button in the Color Gallery.

 Drag-and-drop a color from the Color Line or Color Gallery onto the Edit Color button on the Color Line.
