Coloring objects

When you want to color an object, you have two choices. You can use colors so they each only apply to individual objects (local colors) or you can use named colors so when the named color is changed, all objects using the color reflect the change. Local colors are changed using the Color Editor, named colors are applied using the Color Line (or the Color Gallery).
