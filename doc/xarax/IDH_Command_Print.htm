Print (File Menu) CTRL+P

The Print dialog box is where you control the printing of a document (see Printing documents). It displays basic information on the printer and how the drawing will be printed. If you want to change settings for your printer, click the Printer button. If you want to change how Xara LX will print your drawing, press the Options button. For more information about printing, see the Printing Overview. If you have problems printing with Xara LX, click the Problems? button to open the Printing Troubleshooter.

At the bottom of the dialog box is a brief description of how the document will be printed:

 Layout type - Best Fit, Multiple Fit or Custom Fit (see Print Layout Options).

 The scaling to be used - 100% means print at normal size, 50% means print half size, 200% means print at twice the normal size.

 The orientation - Upright or Sideways.

 How many pages will fit onto the paper with the current settings.
