Message: Only named colors can be spot colors

Cause

You have tried to make a local color a spot color. Only named colors can be spot colors. If you want to make a local color into a spot color, then it must first be given a name.

What Can I Do?

 To create a named spot color, click Make Named. Xara LX will automatically turn this local color into a named spot color for you. You will be asked for a name for the new color.

 To leave the color as it was, click Cancel.
