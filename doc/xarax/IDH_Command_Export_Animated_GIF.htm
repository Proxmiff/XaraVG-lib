Export Animated GIF (File Menu)

Select this item to export all the frames in the Frame Gallery to create an animated GIF file. For more details, see Creating animated GIF files.

Note

 This option is dimmed unless there are at least two frames in the Frame Gallery as you need at least two to create an animation

 You can create animated GIFs using Animation documents. See Drawing and Animation documents for more details.
