To apply a Live Effect

* Select the object or object the usual way.

* Select the Live Effects tool.

* Click the New button on the left side. This displays a categorised list of available plug-ins.

* Select the Effect you want to apply from the list.

This will display a dialog that belongs to the effect. You can adjust the various controls and you usually get a small preview of the effect in the dialog.

* Click OK when you're done. This returns to Xara LX and shows the effect
