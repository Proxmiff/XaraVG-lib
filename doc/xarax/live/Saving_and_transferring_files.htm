Saving and transferring files

When you transfer a file to someone else, unless they have the same Plug-ins installed on their computer they will not see the effects (this is similar the problem of fonts - that the recipient requires to have the same fonts to see an accurate document).  In order to guarantee recipients can see the correct file, lock any the Live Effects you are using.

Because locked effects are saved in the Xara file as a bitmap, this means the files are much larger than for unlocked Live Effects.
