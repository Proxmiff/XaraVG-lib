Adobe Photoshop® vs. Xara Picture Editor plug-ins

The Live Effects tool supports two basic types of plug-in. Photoshop® plug-ins ('old' style and 'new' style) and Xara Picture Editor plug-ins. For instance the Enhance effect listed at the bottom of the list (when you click the New button) is the same tool that appears in the Xara Picture Editor, but can now be used directly on any object in Xara LX. One advantage of XPE plug-ins is that they usually operate directly and interactively on the Xara LX drawing. There is no pop-up preview window. So, for example, you can select the Enhance Live Effect and directly adjust the brightness of the selected object or objects.
