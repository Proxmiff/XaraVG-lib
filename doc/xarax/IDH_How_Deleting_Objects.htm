Deleting objects

You can delete any type of object to remove them from a drawing.

 To delete objects

1. Select the objects (see Selecting objects).

2. From the Edit menu, choose Delete.

Tips

 You can also select the objects then press the Delete key on the keyboard.

 Deleting objects does not put them on the clipboard. If you want to remove an object and leave it on the clipboard, use Cut.

 You can recover deleted objects using Undo. See Undoing and redoing operations.
