Changing the tiling of three color fills

You can change three color fills and four color fills so the colors extend out to infinity or so the color pattern repeats.

 To change the tiling

1. Select the object (see Selecting objects).

2. Choose the Fill Tool.

3. Select Simple or Repeating from the menu on the Fill Tool Infobar.
