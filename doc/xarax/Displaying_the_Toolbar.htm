Displaying the Toolbar

The Toolbar displays all the tools available in Xara LX. Usually it is always displayed but you might accidentally close it

 To display the Toolbar

1. Choose Control bars on the Window menu.

2. Select the check box to the left of Toolbar.

(If the check box is already selected, the Toolbar is hidden from view. Unselect and then select the check box to redisplay the Toolbar.)
