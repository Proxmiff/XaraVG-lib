Web Address (Utilities Menu) CTRL+Shift+W

With this dialog box you can add web addresses to objects in documents so when the document is displayed on a web page, clicking on objects takes you to other pages. For more information, see Adding web addresses to objects. For details on what you can type into this dialog box, see the Web Address Overview.
