Changing the size of a Glow shadow

1. Select the object (see Selecting objects). (See Notes.)

2. Choose the Shadow Tool.

3. Drag on the shadow to change the size.

Notes:

 Glow shadows are directly behind the object so there is no "direction" to change.

 You can also use the Shadow Tool to select objects - more details.

 Changing the Blur also changes the size of the shadow (changing the Blur size).
