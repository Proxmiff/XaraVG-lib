Deleting named colors

Xara LX lets you delete one or more named colors from the Color Gallery and Color Line.

To delete a named color

1. From the Utilities menu, choose Color Gallery.

2. Select the color in the Color Gallery (see Selecting items in a gallery).

3. Click the Delete button in the gallery.

Note

 If any of the selected named colors is in use in the document, you are asked to confirm the deletion. If you choose to delete the used colors, any objects previously using these colors will look the same, but will now have local colors applied.
