Scrollbar

Horizontal and vertical scrollbars let you use the mouse to move the document within the window. Scrollbars are located at the right and bottom edges of the document window. For details on hiding the scrollbars, see Hiding the scrollbars. Scrollbars contain markers, called scroll boxes, that indicate the vertical and horizontal location of the section of document currently in view. For more information on using scrollbars, see Using scrollbars.
