Installing fonts

This program is supplied with a large range of fonts for you to use. Before you can use a font, you must install it. All the installed fonts and all those you can install from the Internet or the CD are shown in the Font Gallery. Fonts come in two formats: TrueType and ATM. To use ATM fonts you may need Adobe Type Manager installed (this depends on your version of Windows). If you are not sure which type to use, we suggest TrueType.

 To install a font

1. Open the Font Gallery.

2. Find and select the required fonts (see Selecting items in a gallery).

3. Click the Install button in the gallery.

Note:

 You cannot use the Fonts Gallery to install or de-install fonts if you have ATM for Windows NT 4. This is an inherent problem with the current version of ATM for Windows NT and beyond the control of Xara LX. Use the standard Windows Control Panel to install and de-install fonts. For details of the Control Panel refer to the Windows documentation.

Tips

 Alternatively, clicking the Apply button installs the font and applies it to the selected text/cursor (or sets the current attribute if nothing is selected).

 Most programs show the installed font(s) in their font list immediately. You may find a few programs that you have to close and then reload before they show the new fonts.
