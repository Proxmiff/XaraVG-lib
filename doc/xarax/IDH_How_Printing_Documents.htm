Printing documents

You often want to print out a document, either to check it looks correct or because it is complete and you want a final, printed copy. For more information on printing, see the Printing Overview.

 To print a document

1. Select the document you wish to print by clicking on it.

2. From the File menu, choose Print.

3. Choose whether to print all objects or just the selected objects.

4. If you are printing a double page spread, choose which pages to print.

5. Choose how many copies to print.

6. Click Print.

Tips

 To select a different printer, click the Printer button in the Print dialog box.

 To set the printing options, click the Options button in the Print dialog box. These options depend on the type of printer selected.

 Printing documents containing transparent objects takes longer than printing documents that do not use transparency.

 If you want to print a drawing on a page the 'wrong' way around, change the orientation options in the Print Layout Options.
