Line Gallery - Dash Pattern

The Dash Pattern section of the Line Gallery allows you to apply different dash patterns to objects outlines.

To change the dash pattern:

 Drag-and-drop the required pattern onto the object. Hold down CTRL to apply the pattern to an object inside another object. See apply inside.

 Double click on the dash pattern to apply it to the selection.

 Click the required dash pattern to select it and click Apply to apply it to the selection.

Note:

 Applying a brush or stroke shape removes any dash pattern already applied to the line.
