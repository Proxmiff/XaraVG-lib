Clipart Gallery - Options Button

Clicking the Options button in the Clipart Gallery opens a menu containing the following options:

 Add Clipart from disk  Add a clipart folder to the Clipart Gallery.

 Empty web clipart cache Empties the cache of downloaded clipart

 Find   Search the Clipart Gallery.

 Sort   Sort the Clipart Gallery.

 Icon size   Change the size of the icons in the Gallery.
