Creating simple text

Simple text objects are the best way to do headings, labels and other small areas of text. You can set the attributes of the text before you type the text (by applying them to the text cursor) or afterwards.

 To create a simple text object

1. Choose the Text Tool.

2. Position the pointer where you want to start the text and click - a red text cursor appears.

3. Set any attributes you want the text to have. For example, color, font, font size.

4. Type the required text, pressing RETURN to create new lines if necessary.
