Changing the type of fill

When you create a shape it is automatically filled with the default fill type. This is usually a flat fill. You can change the fill type at any time.

 To change the type of fill

1. Select the filled object (see Selecting objects).

2. Choose the Fill Tool.

3. Select the new fill type from the drop-down list on the Fill Tool Infobar.

Tip

 You often need to adjust the more complex fill types slightly. For more information see Changing fills.
