Spot Colors can be used in work that will be printed on a litho printing press. Spot colors appear as circles on the Color Line and Color Gallery and each spot color produces an extra separation when you generate color separated output. For simple two and three color jobs you can use spot colors in place of the four process CMYK colors. This usually reduces printing costs. For complex, multicolor jobs, spot colors let you use special inks such a metallic or specific ink colors that CMYK cannot reproduce. Using spot colors like this increases the cost of printing the job. Always talk to your print shop before using spot colors. Note that some printing processes cannot use spot colors.

For details on how to create a spot color, see Creating spot colors.
