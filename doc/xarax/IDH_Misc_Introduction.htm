Welcome to Xara Xtreme LX (Temporary Help)

 

Welcome to Xara Xtreme LX, a fast, easy to use, drawing and illustration program able to create high quality drawings.

 

Xara Xtreme LX, (also still referred to as Xara LX) is based on Xara Xtreme for Windows. All the main tools now work and most features required to create and edit complex drawings are functional and work well.

 

This Help is based on the help files for Xara Xtreme and so is a complete and detailed explanation of how all the tools work. Because Xara Xtreme LX is not yet complete you will find some features described that are not yet implemented.

 

Report Bugs

If you can crash the product please report this, and ideally the exact steps you took to make this happen, to our web-based bug reporting system here

http://bugzilla.xara.com/

If you've an existing Xara user and find anything that doesn't work as it does in the Windows version please also report this.

 

Helpful resources

There is a comprehensive 275 printed manual available. Although it covers the completed, commercial Windows version of Xtreme, you will find it a helpful resource in learning to us Xara LX. It can be ordered from xara.com/products/xtreme/buy.asp.

 

We have an active messageboard of Xara Xtreme users at TalkGraphics.com who will welcome and help new users find their feet. They are happy to answer questions about the product. There is a huge archive of tips, tutorials and answers here, going back years. There are two Xtreme forums.

  • One is generally about how to use Xara Xtreme. This is frequented mostly by users of the Windows version, but the operation of Xara LX is so similar that you will find this a useful resource. http://www.talkgraphics.com/forumdisplay.php?f=16

  • Alternatively there's a specific Xara LX forum. This tends to contains discussions about Linux specific technical issue. http://www.talkgraphics.com/forumdisplay.php?f=41

The Xara Xone at www.xaraxone.com is also a great resource, packed with tutorials and tips. This again has been written for the Windows version but will be extremely helpful in getting to know Xara LX.

 

Help and Contribute

Xara LX is an open source project that would not exist were it not for the efforts of people willing donate their time to help us. Anyone can download the source code and we welcome contributions from anyone to help improve the product, bug fix, add new features or just test and give feedback. You can find out more here

http://www.xaraxtreme.org/developers/

 

Movies

For those new to Xara Xtreme there are some demo movies, including two 'Absolute Beginners' movies that you can download here

http://www.xaraxtreme.org/about/movies.html
