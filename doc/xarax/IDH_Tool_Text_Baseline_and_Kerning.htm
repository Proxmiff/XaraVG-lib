Text Tool - Baseline Shift and Manual Kerning

These two fields on the Text Tool Infobar are used to move characters either vertically (baseline shift) or horizontally (kerning).

Baseline Shift

Changing the baseline shift of text moves it vertically. See Changing the baseline shift of text for more information. You can type values into this field in any unit, e.g. "2cm".

Manual Kerning

Manual kerning spaces characters by placing an invisible 'kern' characters between them. Kerning is measured in ems. For more information, see Kerning text.

Xara LX also has Automatic kerning.

CTRL+ '+' Increase kerning (if the text cursor is present)

CTRL+ '-' Decrease kerning (if the text cursor is present)
