Fill Tool - Bitmap and Fractal Fill Resolution Field

The Resolution field on the Fill Tool Infobar sets the resolution (the size) of the currently selected bitmap/fractal fill. You can change the resolution by:

 Scaling the bitmap/fractal in the fill using the fill handles

 Entering a new value in this field and pressing RETURN.

Note

This field only appears on the Infobar when the selection contains only bitmap/fractal filled objects, and there are no selected fill handles. Click away from the fill handles with the Fill Tool to deselect all the handles.
