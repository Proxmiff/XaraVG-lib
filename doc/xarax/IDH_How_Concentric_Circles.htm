Drawing concentric QuickShapes

Creating concentric circles is simple in Xara LX. Although this describes creating concentric circles, this works equally well for concentric polygons and rectangles.

To draw concentric circles

1. Choose the Ellipse Tool.

2. CTRL-drag to create a circle. (CTRL constrains the ellipse to a circle). See Drawing ellipses and circles.

3. Choose the Contour Tool.

4. Drag one of the eight handles around the object to create the contour. See Creating a contour.

5. Use the Contour Tool to change the number of steps and other settings.
