Deleting colors

Named colors appear on the Color Line and in the Color Gallery. To make this list of colors smaller, you can delete unwanted colors. This is particularly useful where you have created a number of new colors that you no longer want.

Xara LX automatically deletes unused local colors.

 To delete a color

1. From the Utilities menu, choose Color Gallery.

2. In the Color Gallery select the color you want to delete.

3. Click Delete. You are warned if the color is used in the document.

Tip

 You can also select several colors and delete them in one operation. If any of the colors is used in the document, you can either delete only the unused colors or delete all the selected colors.
