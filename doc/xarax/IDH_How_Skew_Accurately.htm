Skewing objects accurately

This method of skewing objects gives you precise control.

 To skew objects accurately

1. Choose the Selector Tool.

2. Select the objects to be skewed.

3. Move the transformation center onto the required horizontal skew axis.

4. Either:

 enter a skew angle (in degrees) in the Skew Field on the Selector Tool Infobar and press RETURN

 or use the Skew field arrow buttons on the Infobar.
