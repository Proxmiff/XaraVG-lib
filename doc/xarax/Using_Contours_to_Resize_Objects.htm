Using Contours to Resize Objects

You can use contours to change the size of objects by making the outline thicker or thinner. This has advantages over resizing for complex shapes like the letter "C":

Left: the original object

Center: resized using a contour

Right: resized by scaling the object up

 To resize using contours

1. Select the object (see Selecting objects) (see Note).

2. Choose the Contour Tool.

3. Give the object a contour of the required size - see Contouring objects for details.

4. Click Inset path on the Contour Tool Infobar. The contour steps disappear and the object is now the size of the contoured object.

Note

 You can also use the Contour Tool to select objects - more details.
