Fill Tool - Fill Type Field

The Fill Type field on the Fill Tool Infobar sets the type of fill applied to the selection.

Options are:

 Flat Fill

 Conical Fill

 Circular Fill

 Linear Fill

 Elliptical Fill

 Diamond Fill

 Three Color Fill

 Four Color Fill

 Bitmap Fill

 Fractal Cloud Fill

 Fractal Plasma Fill

The field shows Many if the selection contains fills of different types.
