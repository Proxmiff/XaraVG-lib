What's Under Development

Xara Xtreme LX is a port of Xara Xtreme to Linux and although it’s now very capable it doesn’t yet include all the functionality of Xtreme. Consequently parts of this temporary help guide refer to tools and functions that aren't currently functional in Xara Xtreme LX. This page gives a quick overview of the main working and non-working parts of the program.. Click here to see a more complete list of working features listed on our web site.

 

Please report any bugs you find as described here. Remember that this help describes all the Xara Xtreme features, some of which are not yet enabled in Xara Xtreme LX. Therefore please don’t report missing features as bugs.

 

Save/Load

Save and load are both fully working, but It is not possible to drag and drop XAR files from an external file browser into LX to open them.

 

Tools

All the major tools work, but some are limited in their functionality. The color editor is fully functional. There’s no support for brushes currently. And there is no Live Effects tool or Xara Picture Editor (XPE).

 

Import/Export

A good selection of the most popular formats are now supported for import and export, including JPEG, GIF, PNG and TIFF.

 

Galleries

The only galleries available are the color, bitmap, layer and line galleries. There is no clipart gallery yet - see /usr/share/xaralx/Examples for some example designs supplied with Xara Xtreme LX.

 

UI and Docking

Xara Xtreme LX uses a completely different window docking mechanism to Xtreme and it is not yet to the required standard. It will be improved soon.

  

Printing

There is some support for printing now included.  Click here for more information.

 

Document info

The document info dialog is not working yet.

  

Movies/updates

The mechanisms for downloading of movies, update of the program etc. are not available yet.
