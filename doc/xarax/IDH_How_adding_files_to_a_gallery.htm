Adding files to a gallery

Once you have added a folder of files to the Clipart Gallery or the Fill Gallery, you can add more files.

 To add files to a gallery

1. Copy the files to the appropriate folder.

2. From the Utilities menu, choose Clipart Gallery or Fill Gallery.

3. Click the Disk Clipart or Disk Fills button (as appropriate) to open the Add Folder dialog box.

4. Browse to the folder.

5. Click the Update button.

Notes

 Clicking Add does not include the new files in the gallery. You must use Update.

 For details on adding folders to these galleries, see Adding clipart to the clipart gallery and Adding fills to the fill gallery.
