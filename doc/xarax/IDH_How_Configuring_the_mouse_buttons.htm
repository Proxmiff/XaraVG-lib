Configuring the mouse buttons

You can configure the left and right mouse buttons so they work in the way you find most comfortable.

 To configure a mouse button

1. From the Utilities Menu, choose Options (see Options).

2. Click the Mouse tab.

3. Use the drop-down lists to choose what you want the left and right mouse buttons to do.

Tip

 If you want to set the mouse buttons back to what they did when you first used Xara LX, click the Reset Defaults button.
