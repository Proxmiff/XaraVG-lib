Changing the transparency profile

You can change the shape of the transition between the start and end of transparency. (This applies to all transparency types except flat, three color and four color.)

 To change the profile of a fill

1. Select the filled object (see Selecting objects).

2. In the Transparency Tool click Profile. This opens the profile dialog box.

3. You either select a predefined profile from the drop-down list or drag the sliders to create a custom profile. The top option in the drop-down list (the straight line) creates a linear transition.

 When creating a custom profile:

 Move the top slider towards:

 -1 for more change at the end of the transparency than at the start.

 +1 for more change at the start.

 Move the bottom slider towards:

 -1 for more change in the center than at the start or end.

 +1 for less change in the center.
