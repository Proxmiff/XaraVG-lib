Hiding the Color Line

The Color Line at the bottom of the Xara LX window is a convenient way of applying colors. However, you can turn off display of the Color Line if you want to maximize the usable area of the window. With the Color Line turned off, you can use the Color Gallery to apply colors.

 To hide the Color Line

In the Window menu, open the Bars submenu and choose Color Line.

Tip

 If you need more screen space, you might like to use full screen mode. See Switching to full screen mode.
