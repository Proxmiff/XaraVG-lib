Clipart Gallery Overview

Shift+F10

The Clipart Gallery lets you easily incorporate clipart from a folder into your document. The folder is usually on the Xara LX CD but you can change to a different drive if necessary. You can also add your own clipart to the gallery if you wish.

Open  Open a new file.

Import  Import a file into the document

Get Clipart Downloads more clipart from the Internet.

Options  Clipart Gallery Options

Remove  Remove the selected sections from the gallery.

Disk Clipart Add a folder of clipart to the gallery from a disk or CD.

Stop Loading Stops downloading clipart from the Internet.

Clipart/Web themes View regular clipart or themed template sets.

Many operations using the Clipart Gallery are made easier by the Clipart Gallery pop-up menu.
