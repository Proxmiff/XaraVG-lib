Changing the bevel size

This describes how to change the size of the bevel at the edge of objects.

 How to change the bevel size:

1. Select the object (see Selecting objects).

2. Choose the Bevel Tool.

3. If necessary apply a bevel (see Applying a bevel).

4. From the Slider Type list choose Size.

5. Either drag the slider or type a value into the text box.

Tip:

 If you have several beveled objects selected, any changes apply to all the bevels.
