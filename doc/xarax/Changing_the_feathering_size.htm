Changing the feathering size

After applying feathering you can change the size of the feather. (You also set the size when applying feathering.)

 To change the feathering size

1. On the Feathering Control Bar either drag the slider or, for precise values, type into the text box.

Note:

 The size of the feather does not change if you scale the object.
