QuickShape Tool - Polygon Button

Pressing the Polygon button on the QuickShape Tool Infobar:

 changes all currently selected QuickShapes to polygons,

 or, if no QuickShapes are currently selected, sets the tool default so the next QuickShape you draw will be a polygon.

The Rectangle Tool can be used for drawing rectangles and squares. It draws QuickShapes, but does not require the setting of fields before the shape can be drawn.
