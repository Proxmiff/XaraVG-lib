Name Layer Dialog Box

The Name Layer dialog box lets you change the name for the layer. Type the name and click OK. For more details, see Renaming layers

Note:

Layers MouseOff, MouseOver, MouseDown, Selected and BackBar are used by the Button & NavBar Tool. We do not recommend renaming these layers or creating layers with these names as this could affect navigation bar creation. For more information on bar creation see Creating Navigation Bars.
