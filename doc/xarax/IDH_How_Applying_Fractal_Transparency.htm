Applying fractal transparency

You can use Fractal Cloud transparency or Fractal Plasma transparency to create the effect of mist or clouds or a random pattern. Each fractal pattern is unique. After applying fractal transparency you can then edit it to create exactly the effect you want.

 To apply fractal fill

1. Select the object (see Selecting objects).

2. Choose the Transparency Tool.

3. Choose Fractal Clouds or Fractal Plasma from the drop-down list on the Transparency Tool Infobar.

For details of how to edit fractal transparency, see Changing fractal transparency.
