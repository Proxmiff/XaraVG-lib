Multi-layer Export

If you have a Xara LX design that has multiple layers, you can now easily export each layer to separate files. This allows you to then import them into another layer-based software package such as Adobe Photoshop and maintain each distinct layer.

You can export the layers as multiple GIF or multiple PNG files. However, it is highly recommended that you export in the PNG format so that you won't be limited to 256 colors, and the results achieved will be significantly better.

 To export multiple layers as PNG files

1. From the File menu, choose Export.

2. In the Export dialog box, choose PNG from the drop-down list.

3. Enter a filename and location and click Save. This opens the PNG Export dialog box where you can change options and preview the bitmap before you save it to disk.

   4. Under Palette Options - Colour Depth choose True color + Alpha to ensure that transparency is activated and that you can import layers properly.

   5. Under the Options panel, ensure that the transparency option is ticked. Then select the Export each layer to a file of its own.

   6. Click Export and your files will be saved.

If you exported your layers with the filename, say, drawing.png, you will get a drawing.png file that shows all layers combined. You will then also get each layer separately as drawing Layer 1.png, drawing Layer 2.png and so on for each layer. You can then import these files into your other software packages.
