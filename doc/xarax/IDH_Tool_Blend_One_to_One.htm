Blend Tool - One to One Button

The one to one setting on the Blend Tool Infobar only has an effect when both objects in the selected blend have the same number of point handles (if the objects are not shapes, the number of point handles they would have if they were converted must be the same).

When the one to one preference is not set, Xara LX will add or removes points, as necessary, to blend one shape to another. When blending shapes with the same number of handles, this can produce unwanted effects. Selecting this button ensures that no points are added or removed during the blend.
