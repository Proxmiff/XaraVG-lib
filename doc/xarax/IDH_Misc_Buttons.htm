List of Default Control Bars

Xara LX contains several default control bars in addition to the Toolbar and the tool-specific Infobar. The button palette contains all the buttons available in Xara LX. You can copy buttons from the button palette to existing or new control bars. You can also remove buttons from control bars (but not from the button palette.)
