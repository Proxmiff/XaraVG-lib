Changing the nudge size

You can change the distance objects and handles move when you nudge them.

 To change the nudge distance

1. From the Utilities Menu, choose Options (see Options).

2. Click the General tab.

3. Enter a nudge distance in the Nudge Size field.

Tip

 You can enter the nudge distance using any units. e.g. '1mm' or '0.2cm'.
