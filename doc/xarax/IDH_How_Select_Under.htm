Selecting objects under other objects

Often objects in the document cover all or most of other objects. Use this to select the hidden objects.

 To select objects under other objects

1. Choose the Selector Tool (see note).

2. Place the mouse pointer above the hidden object.

3. Either:

 ALT-click to select the object underneath,

 or ALT-click again to deselect that object and to select the next object underneath.

The first ALT-click will select the object on top first if it is not selected already. Subsequent ALT-clicks will go on to select the objects underneath. ALT-clicking with the bottom object selected will reselect the top-most object.

 Click to select the object under the selected object. (ALT-click)

Note:

 When selecting navigation bars and rollovers we recommend using the Button & NavBar Tool. See Selecting navigation bar objects.
