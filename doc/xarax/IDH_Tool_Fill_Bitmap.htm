Fill Tool - Bitmap Field

 

The Bitmap field on the Fill Tool Infobar sets the bitmap used in bitmap filled object(s) in the selection. You can preview and select any of the bitmaps used in this document or, by clicking More, select from the fills in the Fill Gallery.

Note

This field only appears on the Fill Tool Infobar when the current Fill Type is set to Bitmap.
