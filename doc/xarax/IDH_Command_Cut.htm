Cut (Edit Menu) CTRL+X or CTRL+DELETE 

Cuts the currently selected objects to the clipboard. The selected objects are removed from the document. Objects on the clipboard can be pasted back any number of times, either into other documents or into other applications. For more information, see Cutting and copying objects to the clipboard.

Tips

 To cut an object to the clipboard quickly, right-click on the object to open the pop-up menu and choose Cut.

 Copy is the same as Cut except that the objects remain in the document.
