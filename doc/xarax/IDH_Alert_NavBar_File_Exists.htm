Message: File already exists

Cause

You have tried to export to an HTML file that already exists.

What Can I Do?

 Click Replace to overwrite the existing file contents.

 Click Insert to insert the graphics into the existing page. This overwrites any existing Xara LX navigation bar or sliced image but does not affect other parts of the page. (See Note.)

 Click Cancel to abandon the export. You can then repeat the operation using a different file name.

Note:

 The graphics are formatted in a <TABLE> tag that is inserted immediately after the <BODY> tag. You may need to move this if the position is wrong - for more details see Creating Navigation Bars (this section also applies to sliced images). If you create the more complicated rollover buttons in a navigation bar, the associated JavaScript is in the <HEAD> tag - you do not need to move this.

 If you are using Macromedia Dreamweaver, see this page.
