Changing the bitmap in a bitmap fill

If an object already has a bitmap fill you can change the bitmap used. It is often easier to change the bitmap by reapplying the bitmap fill (see Applying bitmap fills).

 To change the bitmap in a bitmap fill

1. Select the bitmap filled object (see Selecting objects).

2. Choose the Fill Tool.

3. Select a new bitmap from the drop-down list on the Fill Tool Infobar.

 Or click More to open the Fill Gallery to choose from a wider range of bitmaps.

Tip

 You can also directly use the Bitmap gallery or Fill Gallery. Select the required bitmap and click Fill.
