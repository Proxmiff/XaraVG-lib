Color Editor Menu

Right-click on the Color Editor to open this menu:

Rename Opens the Rename Color dialog box.

Edit Parent Selects the parent (if any) of the current color so it can be edited.

HSV  Selects the HSV color model.

RGB  Selects the RGB color model.

CMYK Selects the CMYK color model.

Grayscale Selects the grayscale color model.

Help  Opens the Xara LX documentation on the Color Editor.
