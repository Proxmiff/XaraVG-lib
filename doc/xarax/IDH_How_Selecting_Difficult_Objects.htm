Selecting difficult objects

If you are having trouble selecting an object, you may like to try the following:

 Zoom in on the object so you can click more accurately. See Zooming in and out.

 Lower the quality setting to outline and click

 If the object is the back object in a blend with many steps, try using a high-zoom on the tiny section of the back object still visible.

If you want to select objects that are on top of other objects, marquee-selection (see Selecting several objects) may move the other objects. Using Shift-drag forces select rather than move.
