Changing the colors in three color fills

After creating a three color fill you can easily change the colors of any of the three handles.

 To change the colors in a three color fill

Simply drag a color from the Color Line or the Color Gallery onto the filled object. When over the object, the shape of the mouse pointer shows which handle the color will be applied to.

Tips

 If you have several objects selected, you can select all fill handles of a particular type (for example all the start handles) by CTRL+Shift-clicking on one of the handles. Use Shift-click to deselect a handle or select an unselected handle.

 You can also apply colors direct to the fill handles. Click on a fill handle to select it and then either:

 click on a color on the Color Line,

 or double-click a color in the Color Gallery,

 or select a color in the Color Gallery and click Apply.

 Caution: if there are no fill handles selected, these methods create a flat fill.
