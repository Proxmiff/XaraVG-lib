Bevel Tool Overview CTRL+F3

Bevels give objects an appearance of depth instead of being flat on the page. Bevels are resolution independent so you can scale them without losing quality.

Note:

We recommend that you apply any bevels before you use any Live Effects. Bevel affects should not be applied after a Live Effect has been used.

Tip:

 Applying a bevel to several objects:

 If the objects are grouped, overlapping bevels merge together. The bevel color is the same as the rear-most object in the group.

 If the objects are ungrouped, each object has a separate bevel. The bevel color is the same as the object color.
