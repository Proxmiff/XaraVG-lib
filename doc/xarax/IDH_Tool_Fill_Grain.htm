Fill Tool - Fractal Fill Grain Field

The Grain field on the Fill Tool Infobar sets the 'grain' for the currently selected object with a Fractal Cloud fill or Fractal Plasma fill applied. Low values gives a smooth, cloud-like effect - higher values give a sharper, more 'grainy' texture.

Note

 This field only appears when the selection contains only fractal filled objects, and there are no selected fill handles. (When a fill handle is selected, this field shows the handle type and color.) Click away from the handles to deselect them.
