Selector Tool - Dimensions Field

The Dimensions Field on the Selector Tool Infobar shows the dimensions of the selection. Its values alter dynamically if you scale or stretch the selection.

You can also resize the selection by:

 Typing into the fields and pressing RETURN,

 Clicking the arrow buttons.

Note

 If the Scale Line Widths button on the Selector Tool Infobar is pressed in, the values in these fields include the line width of objects.
