Button & NavBar Tool - Set new design

Dimmed unless the drawing contains a navigation bar.

This button saves you the trouble of editing each button individually. All you have to do is make the required changes to one button, select it and then click Set new design. This displays the Set Design dialog box. In that dialog box click Set design - this updates all the other buttons to the new design.

Note that the new design can be anywhere on the page - it does not have to be on top of the existing button.

Live stretching affects the operation of Set new design - more details.
