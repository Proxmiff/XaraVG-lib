Subtracting shapes

Subtracting objects removes a section of the underlying objects using the outline of the top-most object.

 To subtract a shape

1. Select the objects (see Selecting objects).

2. On the Arrange menu, choose Combine Shapes then Subtract Shapes (or press CTRL+2).

Tips

 You can use with all object types including bitmaps.
