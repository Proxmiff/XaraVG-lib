Fill Gallery Overview

Shift+F11

The Fill Gallery contains bitmap textures suitable for use as bitmap fills and bitmap transparencies. These bitmaps are not loaded into memory - only when one is used is it loaded.

Fill bitmaps are normally loaded from the Xara LX CD, but you can change to a different drive. You can also add textures to the gallery.

 Import  Import the fill as a bitmap.

 Fill  Apply the bitmap as a bitmap fill.

 Transp  Apply the bitmap as a bitmap transparency.

 Get Fills  Download fills from the Internet.

 Background Set the select fill as the page background.

 Remove  Remove the selected gallery sections.

 Disk Fills  Add more fills to the gallery from your hard drive.

 Options  Fill Gallery options.

 Stop Loading Stop downloading thumbnails.

Many operations using the Fill Gallery are made easier by the Fill Gallery Pop-up Menu.
