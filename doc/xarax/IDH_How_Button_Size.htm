Setting the size of buttons

Buttons on the Toolbar, Infobar and control bars can be small or large. Use the size you prefer.

 

 To set the size of buttons

1. From the Window menu, choose Control Bars.

2. At the bottom of the dialog box, select which bars should have large buttons:

 Top

 Bottom

 Left

 Right

 Floating

Tip

 The Floating option also applies to the buttons in the galleries. If you want larger buttons in the galleries, select this option.
