Message: Cannot open web page

Cause

The program could not open the web page you requested. This may be because:

- The web page does not exist.

- You are not connected to the Internet. Connect to the net and try again.

- The weblink utility that Xara LX uses to open web pages could not be found.

What can I do?

 Ensure the web page exists and ensure your machine is connected to the Internet and try again. If this doesn't solve the problem the weblink utility may have been moved or deleted. Try reinstalling Xara LX.
