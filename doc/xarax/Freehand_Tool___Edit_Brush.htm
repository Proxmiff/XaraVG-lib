Freehand & Brush Tool - Edit Brush

Dimmed when the selected brush pattern is a plain line (as this is non-editable) or when you have two lines selected that use different brushes.

Click this button to open the Edit Brush dialog box. This lets you either edit the brush pattern of the selected line. Optionally you can also create a new brush or edit the existing brush. New brushes appear in to the Select Brush list on the Freehand & Brush Tool Infobar and Brush strokes in the Line Gallery.
