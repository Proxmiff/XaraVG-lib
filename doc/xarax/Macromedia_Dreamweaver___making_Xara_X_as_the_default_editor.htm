Macromedia Dreamweaver - making Xara LX the default editor

Dreamweaver is a popular program for web site design. You can create graphics in Xara LX and then import them into Dreamweaver. You will probably also want to use Xara LX for editing GIFs and JPEGs. The easiest way to do this is to make Xara LX the default graphics editor for Dreamweaver.

 Making Xara LX the default editor

These instructions refer to Dreamweaver 3. Macromedia may introduce new versions for which these instructions do not apply.

1. In Dreamweaver, select Launch external editor on the File menu.

2. Select GIF.

3. Click + in the Editors field.

4. Browse to the Xara LX folder.

5. Select LX.exe

6. Select Make primary to make Xara LX the default editor for GIFs.

You can repeat steps 2 thru 6 for JPEGs and PNGs.
