Viewing document information

You can view a dialog box that gives you a variety of information about the current document.

 To view the document information

1. Ensure the document is the current document. If it is not, click on it.

2. From the File menu, choose Document Info.

Tips

 This dialog box lists all the fonts used in the document. 'Not Installed' by the name of a font means it is not installed on your computer.

 You can type into the Comments field. Comments are saved with the document and are an easy way of including notes with a document
