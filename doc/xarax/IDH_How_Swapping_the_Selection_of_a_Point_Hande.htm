Swapping the selection of point handles

You can swap a point handle on a line or shape between being selected and not selected.

 To swap the selection of point handles

1. Choose the Shape Editor Tool or the Pen Tool (you can also use the Selector Tool - see tip).

2. Shift-click on a point handle on the selected line or shape. If selected the point is deselected, if not selected, the point is selected.

Tips

 You can also use the Selector Tool to swap the selection of point handles if you have enabled edit handles. See Enabling edit handles.
