Making text bold

Once you have created a text object, you can make all the characters in it bold, or just some of them. If you want to create a new text object in bold, just place the text cursor in the document and click the Bold button on the Text Tool Infobar.

 To make text bold

1. Select the text you want to change (see Selecting text).

2. Choose the Text Tool.

3. Click the Bold button on the Infobar.
