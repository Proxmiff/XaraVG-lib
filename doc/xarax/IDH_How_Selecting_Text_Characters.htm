Selecting characters in text objects

This lets you select one character from a line of text.

 To select a character in text objects

1. Choose the Selector Tool.

2. Click on the text object to select it.

3. Either:

 CTRL-click on the character,

 or ALT+CTRL-click on the character twice. The first click selects the line, the second click selects the character.

Tips

 You can also select a character by selecting a region of the text object with only one character in it. See Selecting part of a text object.

 When a text character is selected, the Text Tool Infobar shows the attributes applied to it.

 A selected region of text remains selected even after a tool other than the Text Tool has been chosen. This makes it simple to apply fills and transparency to selected text regions.
