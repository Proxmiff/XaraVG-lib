Message: File is read-only

Cause

The program attempted to write to:

 a file that is read-only,

 a read-only drive, such as a CD-ROM drive.

What Can I Do?

 Use the Windows Explorer to check if the file is read-only. If it is, then the file's read-only status can be changed. (Although there may be a reason why the file is read-only - if in doubt, then contact your System Administrator.) Alternatively, choose another file name or another disk drive.
