Changing the bleed setting

Bleed-off allows your document to be printed over the edge of the page. The standard bleed-off distance is 0.3cm, but If in doubt ask your print shop.

 To change the bleed setting

1. From the Utilities Menu, choose Options (see Options).

2. Click the Page tab.

3. Select the bleed option and enter a value for the bleed size.

Tip

 Note that bleed-off may require a larger paper size and increase costs.
