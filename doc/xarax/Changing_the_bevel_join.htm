Changing the bevel join style

After applying a bevel to an object, you can change the shape of the corners (joins). Options are miter, round and beveled.

 How to change the bevel join style:

1. Select the object (see Selecting objects).

2. Click the appropriate Join button on the Bevel Tool Infobar. Note that changing the join style also changes the joins of any contours and lines in the selected objects.

Tip:

 If you have several beveled objects selected, any changes apply to all the bevels.
