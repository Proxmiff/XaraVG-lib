Deleting guidelines

Deleting an unwanted guideline is easy in Xara LX.

 To delete a guideline

Drag the guideline back onto the appropriate ruler.

Tips

 You can only drag guidelines if the guides layer Editable setting is set. See the Layer Gallery Overview.

 You can also delete a guideline by right-clicking on it, then choosing Delete from the guideline pop-up menu.

 You can also delete a guideline using Properties dialog box in the Layer Gallery. See the Layer Gallery Overview.
