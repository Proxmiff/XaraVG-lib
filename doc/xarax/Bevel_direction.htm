Bevel direction

Bevels can be inside the object or outside. Outside bevels make objects larger; inside bevels do not change the object size.

To change bevel direction:

1. Select the object (see Selecting objects).

2. Choose the Bevel Tool.

3.  If necessary apply a bevel (see Applying a bevel).

4. Click either the Inner or Outer button.

Tip:

 If you have several beveled objects selected, any changes apply to all the bevels.
