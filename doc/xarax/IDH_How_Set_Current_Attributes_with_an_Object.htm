Setting current attributes from an object

You can set the current attributes so they are the same as an object. This is an easy way to create multiple similar objects.

 To set the current attributes from an object

1. Copy the object to the clipboard.

2. Deselect all objects.

3. Choose Paste Attributes from the Edit menu to set the current attributes to those used by the object on the clipboard.

Tip

 If you want to set the attributes for text, choose the Text Tool first.
