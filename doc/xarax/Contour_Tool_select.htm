Selecting objects using the Contour Tool

Before you can apply or edit a contour on an object, you must first select the object. The regular way of selecting objects is to use the Selector Tool but you can also use the Contour Tool.

 Selecting a single object

1. Click on an object to select it. Note that this deselects any other objects that are currently selected.

 Selecting several objects

1. Click on the first object to select it. (Any object can be the first.)

2. Shift-click on the others.

 Shift-click on an object to deselect it.

 Deselecting all objects

1. Click on a blank area of the page.
