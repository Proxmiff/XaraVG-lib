Scaling ellipses

You can scale ellipses easily using the QuickShape Tool or the Ellipse Tool.

 To scale an ellipse

1. Select the ellipse (see Selecting objects).

2. Choose the Ellipse Tool or the QuickShape Tool.

3. Drag a handle on the ellipse.

Tips

 To keep the ellipse the same shape, hold down Shift.

 To resize the ellipse without it rotating or stretching, hold down CTRL.

 You can scale ellipses with the Selector Tool - see Scaling objects.

 You can also resize an ellipse using the drop-down list on the Ellipse Tool Infobar or the QuickShape Tool Infobar. Choose Width and Height from the list and change the values in the fields on the right.
